1. Chạy MinIO (docker)
docker compose -f docker-compose.minio.yml up -d

2. Tạo migration
dotnet ef migrations add InitialCreate

3. Cập nhật database
dotnet ef database update

4. Chạy ứng dụng
dotnet run


Nếu đứng ở thư mục gốc .sln thì chạy lệnh sau:
docker compose -f docker-compose.minio.yml up -d
dotnet ef migrations add InitialCreate -p src\DoAn.HotelParking.Infrastructure -s src\DoAn.HotelParking.Presentation.Api
dotnet ef database update -p src\DoAn.HotelParking.Infrastructure -s src\DoAn.HotelParking.Presentation.Api
dotnet run --project src\DoAn.HotelParking.Presentation.Api


Kiểm tra MinIO:
- API MinIO: http://localhost:9000/minio/health/live
- Console MinIO: http://localhost:9001
- User/Pass mặc định: minioadmin / minioadmin
